# 🧠 Einstein Funny Calculator 😂

A fun and quirky calculator inspired by Einstein! 🚀  
It performs basic calculations while adding a touch of humor and personality to make math more entertaining.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 😂 Funny responses and messages
- 🎨 Simple and user-friendly interface
- ⚡ Fast and lightweight

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 🚀 How to Use

1. Enter two numbers.
2. Choose an operation.
3. Click the calculate button.
4. Let Einstein do the math… with a smile! 😜
